### Git
>建议学习时长：6h  
>权重：⭐⭐⭐⭐  
>理论：⭐  
>实践：⭐⭐⭐⭐  

Git是目前主流的代码版本控制系统，用于代码管理及多人协作。
使用简单，只需记住几个命令就能满足日常需求，IDE中通常也集成了可视化Git操作。

- 基本操作及远程同步（add、commit、push、checkout、pull、merge、reset、revert）
